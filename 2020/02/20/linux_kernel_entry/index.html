Linux kernel入口

2020-02-20
os

入口为init/main.c中的start_kernel函数

经过一系列的初始化，最后调用rest_init()，在这个函数里，最终调用cpu_startup_entry(位于kernel/sched/idle.c)，通过cpu_idle_loop()进入idle线程的循环

从kern.log来看，一开始只有cpu0是启动的，在第一次进入idle之后，开始一个个enable其他的cpu

而module的加载还在后面
